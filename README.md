# Empirical QA Automation Assignment - Aman Pawar

This repository contains automated UI tests for Empirical's QA Assignment using **Selenium**, **TestNG**, and **Gradle**.

---

##  **Assignment Overview**

Automated 2 Test Scenarios:

###  Scenario 1: Sign Up Button Visibility
- Navigate to: [Test URL](https://v0-button-to-open-v0-home-page-h5dizpkwp.vercel.app/)
- Click on the button displayed on the page.
- Assert that the **Sign Up** button is visible on the redirected page.

###  Scenario 2: Search Input Visibility
- Navigate to: [W3Schools Test URL](https://www.w3schools.com/html/tryit.asp?filename=tryhtml_links_target)
- Click on the “Visit W3Schools!” link inside the iframe.
- Assert that the **Search Input Box** is visible on the new page.
